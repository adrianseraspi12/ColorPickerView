# Implementation
Add via Gradle

root build.gradle
```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```
Add the dependecy
```
dependencies {
  implementation 'com.github.adrianseraspi12:ColorPickerView:1.0'
}
```
or via Maven


```<maven>
<repositories>
  <repository>
    <id>jitpack.io</id>
	  <url>https://jitpack.io</url>
	</repository>
</repositories>


<dependency>
  <groupId>com.github.adrianseraspi12</groupId>
	<artifactId>ColorPickerView</artifactId>
	<version>1.0</version>
</dependency>
```
